## 项目经历

### 高并发抽奖 / 秒杀系统

一个用 Go 实现的高并发练习项目，主要用于理解秒杀、抽奖这类瞬时高流量场景。

涉及内容：

- Redis Lua 原子扣减库存
- 用户资格记录，防止重复请求
- MQ 异步削峰
- MySQL 唯一索引保证最终幂等
- 重试、补偿和最终一致性处理
- SSE 实时指标面板，用于观察压测过程

这个项目主要帮助我理解：高并发系统不是简单地“加缓存”，还要考虑超卖、防重、消息丢失、重复消费和数据库压力。

---

### 家校沟通 / IM 平台

一个偏实际业务场景的家校沟通系统，包含小程序端、管理后台、后端服务和服务器部署。

涉及内容：

- WebSocket + HTTP 消息通信
- 游标分页拉取历史消息
- 老师、家长、学生、班级之间的权限关系
- MongoDB 数据建模
- 腾讯云 COS 文件存储
- Docker + Linux + GitHub Actions 部署

这个项目让我更熟悉真实业务系统里的数据关系、权限控制、文件上传、部署流程和线上问题排查。
